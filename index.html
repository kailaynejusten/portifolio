Minha Biblioteca: Uma webpade Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir os serviços e produtos de um pet shop.O objetivo é criar um ambiente virtual onde os clientes possam conhecer os produtos e serviços disponíveis para seus animais de estimação, incluindo informações sobre os itens, suas características e uma opção de compra.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do pet shop e a lista de produtos e serviços em uma seção flexível (.produtos).

Cada produto ou serviço é destacado em seu próprio contêiner (.produto), mostrando uma imagem, descrição, preço e um link de compra.

O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos.

O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência do usuário.

A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora e amigável do site, incentivando os clientes a explorar os produtos e serviços para seus pets.

O projeto é uma aplicação educativa interativa focada em ensinar sobre os cuidados essenciais com os animais de estimação. Utiliza uma série de scripts associados a diferentes atores (sprites) e cenários para criar uma experiência de aprendizado envolvente. O objetivo é familiarizar os usuários com os cuidados necessários para cães, gatos, peixes, pássaros e outros animais, abordando aspectos como alimentação, higiene, brincadeiras e atividades físicas, tudo de maneira divertida e interativa.

Cada ator no projeto possui scripts que definem seu comportamento em resposta a eventos específicos, como o início do programa (quando a bandeira é clicada), cliques dos usuários ou mensagens recebidas. Esses eventos desencadeiam ações como mudanças de cenário, movimentação dos atores, exibição ou ocultação de elementos e a apresentação de diálogos informativos. Por exemplo, ao clicar em um animal, o usuário pode aprender sobre as necessidades daquele pet (como alimentar o peixe ou passear com o cachorro), e o cenário pode mudar para refletir a atividade realizada.

A importância das ferramentas utilizadas, como os blocos de código visual do Scratch, reside na sua acessibilidade e na capacidade de engajar aprendizes de diferentes idades no pensamento computacional, ao mesmo tempo em que explora os cuidados com os animais de forma prática e interativa. O uso de cenários dinâmicos, movimentos de atores e diálogos interativos ajuda a criar uma experiência de aprendizado mais rica e memorável, encorajando os usuários a explorar e interagir com o conteúdo de maneira ativa, promovendo a responsabilidade e a empatia com os animais.
